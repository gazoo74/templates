# dlrun(1)

## NAME

**dlrun**  - example of how to use dl library

## SYNOPSIS

**dlrun** - dlrun [OPTIONS] SYMBOL LIBRARY [--] [ARGS...]

## DESCRIPTION

**dlrun(1)** run a symbol in a shared object or executable.

## OPTIONS

**-h or --help**
	Display this message.

**-V or --version**
	Display the version.
