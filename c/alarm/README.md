# alarm(1)

## NAME

**alarm** - example of how to use alarm function

## SYNOPSIS

**alarm** [OPTIONS] [[HOURS:][MINUTES:]SECONDS]

## DESCRIPTION

**alarm(1)** is a timer based on **alarm(3p)**.

## OPTIONS

**-h or --help**
	Display this message.

**-V or --version**
	Display the version.

## EXIT STATUS

Exits success if timer is reached; exits failure otherwise.
