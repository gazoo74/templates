# ueventcat(1)

## NAME

**ueventcat** - concatenate UEVENT(s) to standard output

## SYNOPSIS

**ueventcat** [OPTIONS]

## DESCRIPTION

Concatenate *UEVENTS(s)* to standard output.

Read netlink socket.

## OPTIONS

**-v or --verbose**
	Turn on verbose messages.

**-D or --debug**
	Turn on debug messages.

**-h or --help**
	Display this message.

**-V or --version**
	Display the version.
