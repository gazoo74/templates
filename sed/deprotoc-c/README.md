# deprotoc-c(1)

## NAME

[deprotoc-c](deprotoc-c.1.adoc) - restore files from nested data.tar.gz archive of a
debian package

## DESCRIPTION

[deprotoc-c(1)](deprotoc-c.1.adoc) can restore individual files from the nested
data.tar.gz archive of a debian package.
